      Ans:

      # Polymorphism:

      Polymorphism means "hewing form".

      It allow different objects to respond the same operation in different way.

      It also allow different ways of using same function depending on what they are operating.

      Eg :- operator over loading.

      Function over loading.

      # Data Encapsulation:-

      The process of bonding /combining together data and function into a single unit is called data encapsulation.

      The data is not accessible to outside and only the member function of the class can access data.

      This insulation of data from direct access by the program is data hiding.

      Encapsulation prevents accidental alteration of the data.

      Data encapsulation is achieve using access controllers.

      Private

      Public

      Protected

      # Data abstraction:-

      It is the process of showing only relevemt data or features of an object excluding other uncessary details.

      In oop , abstraction is achieved by the use of classes where only the essential data and methods one define.

      Abstraction redues the complenify to the user and help them to undestand easily.

      For example: If we represent car as an object, it is sufficient to know possing the brake stops the car. We doesn't need to know actual mechanism process behind the braking system.
