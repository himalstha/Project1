Theory

Q. Define functions, it's types, advantages, components.

Ans:
In the C programming language, a function is a block of code that performs a specific task and can be called multiple times in a program. Functions are defined using the keyword "void" or a specific data type (such as int, float, etc.), followed by the function name and a set of parentheses containing any input parameters.

In C programming, there are several types of functions, including:

1. User-defined functions:
These are functions that are created by the programmer to perform specific tasks. They can take input parameters and return a value or not.

2. Library functions:
These are pre-defined functions in the C library that can be called by the programmer to perform common tasks, such as input/output operations, mathematical calculations, and string manipulation.

3. Recursive functions:
These are functions that call themselves in order to solve a problem by breaking it down into smaller subproblems. Recursion is a powerful technique, but it can also be computationally expensive.

4. Inline functions:
These are functions that are substituted in the code where they are called. The compiler generates a copy of the function code at each point where the function is called, which can improve performance but also increases the size of the final executable.

5. Static functions:
These are functions that are only visible and accessible within the same file they are defined and not available to other files.

6. Extern functions:
These are functions that are defined in another file and can be accessed by other files by using the keyword extern.


The advantages of using functions in C programming include:

1. Reusability:
Functions can be called multiple times in a program, making it easy to reuse the same code for different tasks. This helps to reduce the overall size of the program and makes it more efficient.

2. Modularity: Functions allow for the separation of code into logical units, making it easier to understand and maintain the program.

3. Abstraction:
Functions hide the details of how a task is performed, allowing the programmer to focus on the overall logic of the program.

4. Debugging:
Functions make it easier to locate and fix errors in the program. If an error occurs within a function, it can be isolated and corrected without affecting the rest of the program.

5. Testing:
Functions can be tested independently of the rest of the program, making it easier to ensure that they are working correctly.

6. Function calls can be used for making the program more readable and organized.

7. Function calls can also be used for breaking down a complex problem into smaller subproblems, making it easier to solve.


In C programming, the components of a function include:

1. Function name:
This is the name given to the function, which should be unique and meaningful.

2. Function prototype:
This is the declaration of the function that includes the function's name, return type, and input parameter list. The function prototype is typically placed at the beginning of the program or in a header file.

3. Function parameters:
These are the input variables that are passed to the function when it is called. The data type and number of parameters are specified in the function prototype.

4. Return type:
This is the data type of the value that is returned by the function. The return type is specified in the function prototype. Some functions do not return a value, in that case the return type is void.

5. Function body:
This is the block of code that defines the tasks that the function will perform. The function body is enclosed in curly braces and contains one or more statements.

6. Function call:
This is how the function is executed in the program. The function call consists of the function name followed by the required input parameters in parentheses.

7. Return statement:
This is the statement that ends the execution of the function and returns a value to the calling code. The return statement includes the value or expression to be returned.

In C programming, there are several types of user-defined functions, including:

1. Functions with no return type and no arguments:
These functions do not take any input and do not return any value. They are defined with the keyword "void" and do not have any input parameters.
Example: 

                            
                        
void print_hello()

                            printf("Hello World!\n");

                        }

These functions do not take any input, but return a value of a specific data type. They are defined with the specific data type and do not have any input parameters.
Example: 

                            
                                int get_current_year()

                                }

Example: 

                            
                                float multiply(float a, float b)

                                    return a * b;
                                }

            printf("My name is %s\n", name);
        }

This function takes a character string as input and prints it as "My name is [input]".

Q. Define Call by value and Call by reference with example for each.

Ans:


1. Call by value:
Call by value is a method of passing arguments to a function in which the value of the argument is passed to the function. The function then operates on a copy of the argument, and any changes made to the argument within the function have no effect on the original variable outside of the function.
Example in C: 

        
            #include <stdio.h>
            
                void increment(int x) {
                x++;
                }
            
                int main() {
                int a = 5;
                increment(a);
                printf("%d\n", a); // Output: 5
                }

In this example, the variable 'a' is passed to the 'increment' function by value. The function increments the value of the copy of 'a' within the function, but the original variable 'a' remains unchanged.



2. Call by reference:
Call by reference is a method of passing arguments to a function in which the memory address of the argument is passed to the function. The function then operates on the original variable, and any changes made to the argument within the function are reflected in the original variable outside of the function.
Example in C: 

    
        #include <stdio.h>
        
            void increment(int *x) {
            (*x)++;
            }
        
            int main() {
            int a = 5;
            increment(&a);
            printf("%d\n", a); // Output: 6
            }

In this example, the memory address of the variable 'a' is passed to the 'increment'function by reference. The function increments the value of the original variable 'a' within the function, and the new value is reflected in the original variable outside of the function.
